    1. Eventos del ratón.
    1.1 Eventos de puntero
    mousemove

    - El evento mousemove ocurre cuando se mueve el ratón mientras esta sobre un elemento.
    Este evento se produce continuamente, mientras el ratón permanece dentro del elemento.

    mouseout

    - El evento mouseout ocurre cuando el ratón está dentro de un elemento y se mueve fuera de ese elemento, o fuera de uno de sus elementos secundarios.

    mouseleave

    - El evento mouseleave ocurre cuando el ratón está dentro de un elemento y se mueve fuera de ese elemento, pero no a sus elementos secundarios
    (no burbujea: no se propaga por la jerarquía del documento).

    mouseover

    - El evento mouseover ocurre cuando el ratón esta fuera de un elemento y se mueve a dentro de ese elemento, o a uno de sus elementos secundarios.

    mouseenter

    - El evento mouseenter ocurre cuando el ratón esta fuera de un elemento y se mueve a dentro de ese elemento, pero no a sus elementos secundarios
    (no burbujea: no se propaga por la jerarquía del documento).

    1.1 Eventos de click
    mousedown

    - Mousedown ocurre cuando se “pincha/presiona” un botón del ratón sobre un elemento.

    mouseup

    - El evento mouseup ocurre cuando se suelta el botón del ratón, que previamente había sido pulsado sobre un elemento.

    click

    - El evento click ocurre cuando se hace click, generalmente, con el botón izquierdo del ratón sobre el elemento.

    dblclick

    - El evento dblclick ocurre cuando se hace doble click sobre el elemento.

    contextmenu

    - El evento contextmenu,contextmenu, ocurre cuando se hace clic derecho en un elemento para abrir el menú contextual.

    Aunque el evento es compatible con todos los navegadores, el atributo “actualmente” solo es compatible con Firefox.
    1.1 Eventos de la rueda
    wheel

    El evento wheel ocurre cuando la rueda del mouse se desplaza hacia arriba o hacia abajo sobre un elemento.

    El evento wheel también ocurre cuando el usuario se desplaza o se acerca o aleja de un elemento mediante el uso de una pantalla táctil (como el "mouse" de un ordenador portátil).
    2. Eventos del teclado.
    keydown

    - El evento keydown ocurre cuando se presiona cualquier tecla del teclado, independientemente de que se suelte o no se suelte a continuación.

    Si la mantenemos pulsada el evento se repetirá una y otra vez hasta que soltemos la tecla.
    keypress

    - El evento keypress ocurre cuando se mantiene pulsada una tecla del teclado.

    Antes de este evento se produce un keydown en el momento que se pulsa la tecla.
    El evento keypress no se activa para todas las teclas (por ejemplo, ALT, CTRL, SHIFT, ESC, barra espaciadora, etc.) en todos los navegadores.
    Para detectar solo si el usuario ha presionado una tecla, se recomienda el uso de keydown.
    keyup

    - El evento keyup ocurre cuando se suelta, una tecla, que previamente había sido pulsada.
